ISMG 2050 • Week 8 Course Map

Exploring Data Tools, Data Types, and Illustrations

Week 8 pulls together consolidation, linked workbooks, data types, and visual enhancements so students can build polished, multi-sheet workbooks that connect data sources and communicate clearly.

Content Focus

Weekly Topics

Data Tools & Enhancements
  • Consolidate data by position in a workbook
  • Consolidate data by category in a workbook
  • Link workbooks with the Consolidate command and with formulas
  • Format values using built-in and custom formats
  • Explore and work with linked data types
  • Insert illustrations in a worksheet
  • Use hyperlinks in a worksheet
  • Set workbook security by marking as final or setting a password

Application

Assignments

Realistic Scenarios
Independent Project – Wilson Home Entertainment Systems
Cash flow & consolidated view
  • Monitor cash flow at each location and consolidate data
  • Insert hyperlinks to supporting worksheets from a summary sheet
  • Insert an image and work on a vendor list
  • Apply appropriate formatting and, if needed, workbook protection
Advanced Project – American River Cycling Club
Race day data & visuals
  • Format and consolidate data for four states
  • Insert the club’s logo in a header
  • Create a SmartArt graphic about race preparation tips
  • Use hyperlinks or illustrations to enhance navigation and communication
Challenge Project – Timesheet Consolidation
Employee hours workbook
  • Create a timesheet workbook with one sheet per employee or time period
  • At week’s end, consolidate the sheets to calculate total hours per employee
  • Use formulas or Consolidate tools to link data across sheets
  • Optionally secure the workbook or mark final for distribution
Student Differentiators

Competencies

What Students Can Do
Competency 1

Integrate and consolidate data across multiple sheets and workbooks using the Consolidate command, formulas, linked data types, and hyperlinks to create coherent summary views for complex business scenarios.

Competency 2

Enhance workbooks with professional formatting, custom number formats, illustrations, SmartArt, and basic security settings so that final files are visually polished, easier to navigate, and appropriate for sharing with stakeholders.

Week 8 emphasizes “final polish”: consolidating across workbooks, enhancing visuals, and securing deliverables as students prepare to share Excel solutions beyond the classroom.
